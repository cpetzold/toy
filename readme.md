Toy is a 2d nodejs game library.  It utilizes javascript bindings to sdl2.

## Usage

### Install SDL2

Download from [http://www.libsdl.org/hg.php](http://www.libsdl.org/hg.php)

```bash
./configure
make
sudo make install
```

### Install toy

```bash
npm install toy
```
